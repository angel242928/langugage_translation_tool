A simple Python Language Translation Tool built using Tkinter and the deep-translator library. It allows users to translate text between multiple languages through a graphical user interface.

## Features
- Translate text between different languages
- Simple and user-friendly GUI
- Multiple language support
- Built with Python and Tkinter

## Technologies Used
- Python
- Tkinter
- deep-translator

## Installation

bash
pip install deep-translator


## Run the Project

bash
python app.py
